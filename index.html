What this is

A working catalog, not a manual.

Skills are small, focused tools that extend Claude with the specific workflows we run inside Titus. Document creation, candidate evaluation, partner intelligence, and the pieces of operations that used to be manual.

This page is the source of truth for what is live, what is coming, and how to ask for something new. If a skill is not listed here, it is not part of the supported library.

How to contribute

Pitch a skill, get it built.

Anyone at Titus can build personal skills for their own use. Skills published to the library are reviewed, polished, and supported.

01

Personal vs. team skills

Build whatever helps you personally, no approval needed. The library is curated: skills here have to work reliably for more than one person.

02

What makes a good candidate

  • Recurring task, not a one-off
  • Multiple people would use it
  • Clear inputs and outputs
  • Manual workflow exists today
03

How to propose one

DM Koal in Slack with four things: the pain point, your current manual workflow, how often you do it, and who else would use it.

04

What happens next

Review within a week. If it's a fit, we scope, build, polish, and publish, with attribution. If not, you'll get a clear reason and any adjacent skill that already covers it.
